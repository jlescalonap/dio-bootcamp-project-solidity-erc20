# Projeto DIO Coin (versão aula)

Esse projeto é a prática de como criar um token **ERC20** na rede Ethereum, que foi realizada junto com o professor **Cassiano** da ***DIO***.

> Tem alguns bus que irão ser resolvidos nas proximas versões pois na > video aula não ficaram explicados, por exemplo o professor executou > normalmente a função do contrato transferFrom pra qual tinha que > delegar uma quantidade de tokens para ser transferido desde uma > terceira conta, mas ao tentar executar essa função, se presenta um > problema de "Gas" no editor, mesmo assim foi investigado e existem > multiples razões pelas quais poderia estar dando esse erro, então > futuramente estarei resolvendo esse issue, pode ser desde a versão > utilizada no compilador como poderia também ser algum erro na > construção das funções.
